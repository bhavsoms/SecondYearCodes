# AoA
Analysis of Algorithms (AoA) course.

Algorithms Included:

### Sorting Algorithms:
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort
### Dynamic Programming:
- Knapsack Problem
- Graph Algorithms:
- Minimum Spanning Tree (Prim's 
- Algorithm or Kruskal's Algorithm)
- Traveling Salesman Problem (heuristic approach)
### Backtracking:
- N-Queens Problem
### String Matching:
- Knuth-Morris-Pratt (KMP) Algorithm
